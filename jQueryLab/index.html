Internet Engineering Task Force

The IETF is an international group of network designers, operators, vendors, and researchers dedicated to the evolution of Internet architecture. The organization produces documents that outline influence the way the Internet is designed, used, and managed. One specific issue the IETF involves itself with is privacy and security on the Inernet.

Internet Corporation for Assigned Names and Numbers

ICANN is a nonprofit dedicated to coordinating the maintenance of databases related to namespaces and numerical spaces on the Internet. One specific effort ICANN coordinates is managing IP addresses so that there are no identical addresses.

World Wide Web Consortium

The W3C is an international community dedicated to establishing web standards. For example, the organization supports web design that allows all people to access webpages, with standards to ensure those with different disabilities can still navigate the webpages.
